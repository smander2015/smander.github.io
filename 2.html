Shadowsocks服务端配置 安装： yum install python yum install python-pip pip install shadowsocks 配置： vi /etc/shadowsocks.json 在其中写入： { "server":"myserverip", //0.0.0.0 "server_port":8989, "local_address": "127.0.0.1", "local_port":1080, "password":"mypassword", "timeout":300, "method":"aes-256-cfb", "fast_open": false, "workers": 1 } 启动服务器： ssserver -c /etc/shadowsocks.json 后台运行服务器： nohup ssserver -c /etc/shadowsocks.json > /dev/null 2>&1 & 关闭： killall ssserver
