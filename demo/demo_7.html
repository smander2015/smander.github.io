软件破解常规流程案例演示


        第四步注册机：


        由于补丁法不能适用于全平台，故需要编写注册机来批量注册。
        首先用C语言来模拟之前用户名验证算法F1和序列号验证算法F2的过程： 

			int F1(char* username){
				int sum = 0;
				for(int i = 0; username[i] != '\0';i++){
					if(username[i] < 'A')
						break;
					sum += username[i] >= 'Z'?(username[i] - 0x20):username[i];
				}
				return sum^0x5678;
			}

			int F2(char* serial){
				int sum = 0;
				for(int i=0;serial[i]!='\0';i++){
					sum *= 0xA;
					sum += serial[i]-0x30;
				}
				return sum^0x1234;
			}


        对F2算法求逆生成注册机，对于每个用户名自动生成注册码： 

			int keygen(char* username){
				int sum = 0;
				for(int i = 0; username[i] != '\0';i++){
					if(username[i] < 'A')
						break;
					sum += username[i] >= 'Z'?(username[i] - 0x20):username[i];
				}
				return sum^0x5678^0x1234;
			}


        主程序： 

			int main(){
				char username[10];
				printf("Please input username:");
				scanf("%s",username);
				printf("\nserial:%d",keygen(username));
				return 0;
			}


        运行结果：


        以此方式可以批量生成用户名和注册码，实现软件的破解。
